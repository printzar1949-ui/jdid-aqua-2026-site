# 🌊 AquaVia - أكوافيا

AquaVia هو مشروع بيئي مبتكر يهدف إلى إنتاج مياه نظيفة من الهواء باستخدام تقنيات التكثيف والتنقية.

## 💡 فكرة المشروع
تحويل الرطوبة في الهواء إلى مياه صالحة للشرب باستخدام تقنيات مستدامة.

## 🌍 الأهداف
- توفير مياه نظيفة
- حل مشكلة شح المياه
- دعم البيئة
